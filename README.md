# DuoUpdater Releases

This repository is the public binary distribution channel for DuoUpdater.

- It contains notarized macOS release artifacts and release notes.
- It does not contain source code.
- Download the latest build from the [Releases](https://github.com/jizhi0v0/duo-updater-releases/releases) page.

## Install

1. Download the latest `DuoUpdater-*-macos.zip` asset from Releases.
2. Unzip it and move `DuoUpdater.app` to `/Applications`.
3. Open it once and approve any macOS permission prompts if needed.

## Notes

- Releases are signed with Developer ID and notarized by Apple.
- The source repository is maintained separately and is not published here.
